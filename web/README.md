
# Kid Healthcare System (React + Bootstrap)

Responsive React implementation based on the provided desktop screenshots and videos.

## Included pages
- Landing page (`/`)
- Registration form (`/registration`)
- Consent form (`/consent`)

## Behaviors implemented
- Fixed navbar
- Smooth anchor scrolling on landing page
- `Contact` jumps to `Ourselves`
- `About` jumps to `Your Problem / Our Solution`
- Auto-playing + manual carousel
- Ourselves tab switching (`Our Team`, `Goal`, `Laboratory`)
- Responsive layout for desktop, tablet, and mobile
- Hover lift/shadow on cards
- Form focus states

## Run locally
```bash
npm install
npm run dev
```

## Notes
- This version is built from screenshots and the provided videos, so effects that were not visible were recreated with reasonable React/CSS behavior.
- Bootstrap is used for layout helpers and forms; pixel-perfect details are handled with custom CSS.
